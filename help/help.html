Usage: help [command]

Help prints online documentation on a given command or topic. This is done by opening a file with the commands name plus the extension ".html". Help returns with an error, if this file can not be found in the current directory. This version of help only support a very small subset of XHTML tags, it even lacks support of hyperlinks.
